## 颜色方案

### 状态颜色
- **工作时间**: 红色 `RGB(220, 38, 38)` - #DC2626
- **短休息**: 绿色 `RGB(34, 197, 94)` - #22C55E  
- **长休息**: 蓝色 `RGB(59, 130, 246)` - #3B82F6

### 按钮状态
- **启用**: 白色背景，黑色文字，可点击
- **禁用**: 灰色背景，灰色文字，不可点击

## 如何在 Windows 上运行和测试

### 方法1：从源代码运行
```bash
cd PomodoroTimer/PomodoroTimer
dotnet run
```

### 方法2：运行编译后的程序
1. 导航到 `PomodoroTimer/bin/Release/net6.0-windows/`
2. 双击 `PomodoroTimer.exe`

### 系统要求
- Windows 7 或更高版本
- .NET 6.0 Runtime 或更高版本
