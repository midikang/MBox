🍅 PomodoroTimer 应用界面展示

1. 初始状态 - 工作模式

特点：
  • 状态显示为"工作时间"（红色）
  • 倒计时设置为 25:00
  • 进度条为空（0%）
  • "开始"和"重置"按钮启用
  • "暂停"按钮禁用（灰色）
  • 完成的番茄钟计数为 0

2. 运行中状态

特点：
  • 窗口标题显示剩余时间："18:32 - 番茄钟"
  • 倒计时正在进行：18:32（每秒递减）
  • 进度条实时增长，约 60% 完成
  • "开始"按钮禁用，"暂停"和"重置"按钮启用
  • 所有模式选择按钮禁用（运行时无法切换模式）
  • 倒计时结束后会播放提示音并弹出提醒

3. 短休息模式

特点：
  • 状态显示为"短休息"（绿色）
  • 倒计时设置为 05:00（5分钟）
  • 进度条重置为 0%
  • 完成的番茄钟计数增加到 1
  • 工作时段完成后自动切换到此模式
  • 提供短暂休息时间以保持专注

4. 长休息模式

特点：
  • 状态显示为"长休息"（蓝色）
  • 倒计时设置为 15:00（15分钟）
  • 完成的番茄钟计数为 4
  • 完成 4 个工作番茄钟后自动建议此模式
  • 提供更长的休息时间以充分恢复精力
  • 遵循标准番茄工作法流程

应用功能总结

🎯 核心功能

  • 三种计时模式：工作（25分钟）、短休息（5分钟）、长休息（15分钟）
  • 智能提醒：倒计时结束时播放系统提示音并弹出对话框
  • 进度追踪：实时进度条和完成番茄钟计数
  • 自动建议：完成4个工作番茄钟后自动建议长休息
  • 灵活控制：开始、暂停、重置，随时掌控节奏

🎨 界面设计

  • 颜色编码：红色（工作）、绿色（短休）、蓝色（长休）
  • 大号显示：48pt 粗体倒计时，清晰易读
  • 简洁布局：固定窗口大小（384x391px），无干扰元素
  • 即时反馈：窗口标题、进度条、按钮状态实时更新

💡 使用建议

  • 工作时段：关闭所有通知，专注于单一任务
  • 短休息：离开电脑，做简单的伸展运动
  • 长休息：完全放松，散步或做其他活动
  • 记录追踪：可以用纸笔记录每个番茄钟完成的任务

⚙️ 系统要求与运行

系统要求

  • Windows 7 或更高版本
  • .NET 6.0 Runtime 或更高版本
  • 最小分辨率：800x600

运行方法

# 方法1: 从源代码运行
cd PomodoroTimer/PomodoroTimer
dotnet run

# 方法2: 运行编译后的程序
cd PomodoroTimer/bin/Release/net6.0-windows
./PomodoroTimer.exe
