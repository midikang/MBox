# PomodoroTimer 应用截图

## 应用界面展示

由于番茄钟是 Windows Forms 应用程序，以下是应用在不同状态下的界面预览图（SVG 格式）。

---

## 1. 初始状态 - 工作模式

![初始工作模式](01-initial-work-mode.svg)

**特点：**
- 状态显示为"工作时间"（红色 #DC2626）
- 倒计时设置为 25:00
- 进度条为空（0%）
- "开始"和"重置"按钮启用
- "暂停"按钮禁用
- 所有模式选择按钮启用
- 完成的番茄钟计数为 0

---

## 2. 运行中状态

![运行中状态](02-running-state.svg)

**特点：**
- 窗口标题显示剩余时间："18:32 - 番茄钟"
- 倒计时正在进行：18:32（每秒递减）
- 进度条实时增长，约 60% 完成
- "开始"按钮禁用
- "暂停"和"重置"按钮启用
- 所有模式选择按钮禁用（运行时无法切换模式）
- 完成的番茄钟计数为 0（工作未完成）

---

## 3. 短休息模式

![短休息模式](03-short-break-mode.svg)

**特点：**
- 状态显示为"短休息"（绿色 #22C55E）
- 倒计时设置为 05:00（5分钟）
- 进度条重置为 0%
- 完成的番茄钟计数增加到 1
- 按钮状态恢复初始设置
- 工作时段完成后自动切换到此模式

---

## 4. 长休息模式

![长休息模式](04-long-break-mode.svg)

**特点：**
- 状态显示为"长休息"（蓝色 #3B82F6）
- 倒计时设置为 15:00（15分钟）
- 进度条重置为 0%
- 完成的番茄钟计数为 4
- 完成 4 个工作番茄钟后自动建议此模式
- 提供更长的休息时间以充分恢复

## 功能演示

### 完成提醒
计时结束时会：
1. 播放系统提示音（Beep）
2. 弹出消息对话框：
   - 工作结束："工作时间结束！该休息一下了。"
   - 短休息结束："短休息结束！准备继续工作。"
   - 长休息结束："长休息结束！准备继续工作。"
3. 自动切换到建议的下一个模式

### 智能建议逻辑
```
工作 → 短休息 → 工作 → 短休息 → 工作 → 短休息 → 工作 → 长休息
 ↓       ↓       ↓       ↓       ↓       ↓       ↓       ↓
计数+1  不计数  计数+1  不计数  计数+1  不计数  计数+1  不计数
```

当完成 4 个工作番茄钟时，自动建议长休息（15分钟）

## 如何运行

### 在 Windows 系统上
```bash
# 方法1: 从源代码运行
cd PomodoroTimer/PomodoroTimer
dotnet run

# 方法2: 运行编译后的程序
cd PomodoroTimer/bin/Release/net6.0-windows
./PomodoroTimer.exe
```

### 系统要求
- Windows 7 或更高版本
- .NET 6.0 Runtime 或更高版本

## 实际使用建议

1. **保持专注**: 工作时段关闭所有通知和干扰
2. **严格遵守**: 不要在工作时段处理其他任务
3. **真正休息**: 休息时段离开电脑，做伸展运动
4. **持续改进**: 记录每个番茄钟完成的任务，反思效率
