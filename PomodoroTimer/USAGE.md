# 番茄钟使用指南 / PomodoroTimer Usage Guide

## 应用界面说明 / Application Interface

### 主界面组成
1. **状态标签** - 显示当前模式（工作时间/短休息/长休息）
2. **倒计时显示** - 大字体显示剩余时间（分:秒）
3. **进度条** - 可视化显示当前时段进度
4. **完成计数** - 统计已完成的番茄钟数量
5. **模式选择按钮** - 快速切换不同时间模式
6. **控制按钮** - 开始、暂停、重置计时器

## 使用步骤 / Usage Steps

### 第一次使用
1. 启动 `PomodoroTimer.exe`
2. 默认进入"工作时间"模式（25分钟）
3. 点击"开始"按钮开始计时
4. 专注工作直到计时器结束
5. 听到提示音并看到完成提示
6. 根据建议进入休息模式

### 日常使用流程

#### 标准番茄工作法流程
```
工作25分钟 → 短休息5分钟 → 工作25分钟 → 短休息5分钟 
→ 工作25分钟 → 短休息5分钟 → 工作25分钟 → 长休息15分钟
```

#### 操作说明
1. **开始工作**
   - 确保当前是"工作时间"模式
   - 点击"开始"按钮
   - 专注工作，避免中断

2. **暂停计时**
   - 如需临时中断，点击"暂停"按钮
   - 处理完紧急事务后，点击"开始"继续

3. **重置计时**
   - 如需重新开始当前时段，点击"重置"按钮
   - 计时器会重置到当前模式的初始时间

4. **切换模式**
   - 停止或重置计时器后可切换模式
   - 点击"工作 25分"/"短休 5分"/"长休 15分"按钮
   - 计时中无法切换模式

## 功能特性 / Features

### 智能提醒
- ✅ 计时结束时播放系统提示音
- ✅ 弹出提醒对话框
- ✅ 窗口标题实时显示剩余时间

### 智能建议
- ✅ 工作时段结束后，自动建议休息
- ✅ 完成4个番茄钟后，建议长休息

### 进度追踪
- ✅ 实时显示当前时段进度
- ✅ 统计已完成的工作番茄钟数量

  - 红色：工作时间
  - 绿色：短休息
  - 蓝色：长休息

## 最佳实践 / Best Practices

### 工作时段 (25分钟)
- ✅ 关闭所有通知和干扰
- ✅ 专注于单一任务
- ✅ 不要中途切换任务
- ✅ 如果任务完成，可以回顾或优化

### 短休息 (5分钟)
- ✅ 离开电脑
- ✅ 做简单的伸展运动
- ✅ 喝水、休息眼睛
- ❌ 不要查看邮件或社交媒体

### 长休息 (15分钟)
- ✅ 完全放松
- ✅ 散步或做其他活动
- ✅ 吃点小食
- ✅ 为下一轮工作充电

## 常见问题 / FAQ

### Q: 能否自定义时间？
A: 当前版本使用标准番茄工作法时间设置。如需自定义，需要修改源代码中的常量。

### Q: 计时器会锁定屏幕吗？
A: 不会。计时器在后台运行，不影响其他应用使用。

### Q: 如何关闭应用？
A: 直接关闭窗口即可。下次启动会重置所有计数。

### Q: 数据会保存吗？
A: 当前版本不保存数据。关闭应用后，完成的番茄钟计数会清零。
