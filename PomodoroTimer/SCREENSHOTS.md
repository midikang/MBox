# PomodoroTimer 应用截图说明

## 应用外观

由于这是一个 Windows Forms 应用程序，需要在 Windows 系统上运行才能查看实际界面。以下是应用的主要视觉特征描述：

## 主界面特征

### 窗口外观
- 固定大小窗口 (384x391 像素)
- 标准 Windows 窗口边框
- 居中显示在屏幕中央
- 标题栏显示 "番茄钟 - Pomodoro Timer"

### 界面元素（从上到下）

#### 1. 状态显示区域
```
┌─────────────────────────┐
│       工作时间          │  <- 红色文字
└─────────────────────────┘
```
- 14号 Arial 字体
- 居中对齐
- 根据模式变色（红/绿/蓝）

#### 2. 倒计时显示
```
┌─────────────────────────┐
│                         │
│        25:00            │  <- 48号粗体
│                         │
└─────────────────────────┘
```
- 超大号数字显示
- 清晰易读
- 格式: MM:SS

#### 3. 进度条
```
████████████░░░░░░░░░░░░░  <- 横向进度条
```
- 标准 Windows 进度条样式
- 宽度占满大部分窗口
- 从左到右填充显示进度

#### 4. 完成统计
```
完成的番茄钟: 0
```
- 小号字体
- 居中显示
- 实时更新计数

#### 5. 模式选择按钮
```
┌──────────┐  ┌──────────┐  ┌──────────┐
│工作 25分 │  │短休 5分  │  │长休 15分 │
└──────────┘  └──────────┘  └──────────┘
```
- 三个等宽按钮
- 水平排列
- 11号字体

#### 6. 控制按钮
```
┌──────────┐  ┌──────────┐  ┌──────────┐
│  开 始   │  │  暂 停   │  │  重 置   │
└──────────┘  └──────────┘  └──────────┘
```
- 三个等宽按钮
- 水平排列
- 12号字体
- 暂停按钮初始为灰色（禁用）

## 不同状态下的界面

### 工作模式（默认）
- 状态文字: "工作时间" (红色)
- 倒计时: 25:00
- 开始按钮: 启用（白色背景）
- 暂停按钮: 禁用（灰色）

### 短休息模式
- 状态文字: "短休息" (绿色)
- 倒计时: 05:00
- 按钮状态同上

### 长休息模式
- 状态文字: "长休息" (蓝色)
- 倒计时: 15:00
- 按钮状态同上

### 运行中状态
- 倒计时数字不断减少
- 进度条不断增长
- 窗口标题显示剩余时间: "MM:SS - 番茄钟"
- 开始按钮: 禁用（灰色）
- 暂停按钮: 启用（白色背景）
- 所有模式按钮: 禁用（灰色）

## 提醒对话框

当计时结束时，会弹出标准 Windows 消息框：

```
┌─────────────────────────────────┐
│  番茄钟提醒              [i]     │
├─────────────────────────────────┤
│                                  │
│  工作时间结束！该休息一下了。    │
│                                  │
│              ┌──────┐            │
│              │  OK  │            │
│              └──────┘            │
└─────────────────────────────────┘
```

消息内容根据模式变化：
- 工作结束: "工作时间结束！该休息一下了。"
- 短休息结束: "短休息结束！准备继续工作。"
- 长休息结束: "长休息结束！准备继续工作。"

## 如何运行并查看界面

### 在 Windows 上运行

1. **通过源代码运行**:
   ```bash
   cd PomodoroTimer/PomodoroTimer
   dotnet run
   ```

2. **运行编译后的程序**:
   - 导航到: `PomodoroTimer/bin/Release/net6.0-windows/`
   - 双击 `PomodoroTimer.exe`

### 系统要求
- Windows 7 或更高版本
- .NET 6.0 Runtime 或更高版本
- 800x600 或更高分辨率显示器

## 实际运行效果

应用启动后：
1. 窗口立即出现在屏幕中央
2. 显示默认的工作模式界面
3. 所有元素整齐排列，间距适中
4. 按钮大小适中，易于点击
5. 字体大小适中，易于阅读
6. 整体界面简洁清爽，专注于功能

## 用户体验亮点

- ✅ 简洁的设计，无干扰
- ✅ 大号倒计时，一目了然
- ✅ 清晰的视觉反馈（颜色、进度条）
- ✅ 友好的提醒消息
- ✅ 智能的模式切换建议
- ✅ 完成统计激励用户

## 注意事项

- 本应用是原生 Windows Forms 应用
- 需要在 Windows 系统上运行
- 在 Linux/Mac 上无法直接运行（需要 Wine 等模拟器）
- 如需跨平台支持，可考虑使用 Avalonia UI 或 MAUI 重写
