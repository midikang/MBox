# PomodoroTimer - 番茄钟

一个简单实用的番茄工作法计时器，使用 C# 和 Windows Forms 开发。

## 功能特点

- 🍅 标准番茄工作法计时
  - 25分钟工作时间
  - 5分钟短休息
  - 15分钟长休息
- ⏱️ 清晰的倒计时显示
- 📊 实时进度条
- 🔔 完成提醒（系统声音 + 弹窗提示）
- 📈 自动统计完成的番茄钟数量
- 🎯 智能推荐下一个时段（工作4个番茄钟后推荐长休息）
- ⏯️ 开始、暂停、重置控制

## 什么是番茄工作法？

番茄工作法（Pomodoro Technique）是一种时间管理方法，由Francesco Cirillo在1980年代末期创立。它使用定时器将工作分解为多个时间段（通常为25分钟），每个时间段称为一个"番茄钟"，中间穿插短暂休息。

**标准流程：**
1. 选择要完成的任务
2. 设置番茄钟为25分钟
3. 专注工作直到计时结束
4. 休息5分钟
5. 每4个番茄钟后，休息15-30分钟

## 使用方法

1. **选择时间模式**
   - 点击"工作 25分"开始工作时段
   - 点击"短休 5分"开始短休息
   - 点击"长休 15分"开始长休息

2. **控制计时器**
   - 点击"开始"按钮启动计时器
   - 点击"暂停"按钮暂停计时器
   - 点击"重置"按钮重置当前时段

3. **查看进度**
   - 界面顶部显示当前模式（工作时间/短休息/长休息）
   - 中间显示剩余时间
   - 进度条显示当前时段进度
   - 底部显示已完成的番茄钟数量

4. **完成提醒**
   - 计时结束时会播放系统提示音
   - 弹出提醒窗口
   - 自动切换到建议的下一个时段

## 系统要求

- Windows 操作系统
- .NET 6.0 Runtime 或更高版本

## 编译说明

```bash
cd PomodoroTimer
dotnet restore
dotnet build -c Release
```

编译后的程序位于：`PomodoroTimer/bin/Release/net6.0-windows/`

## 运行

在 Windows 系统上直接运行 `PomodoroTimer.exe`

## 使用技巧

1. **保持专注**：在工作时段关闭所有干扰，专注于单一任务
2. **严格休息**：休息时间要真正放松，离开工作环境
3. **记录任务**：可以记录每个番茄钟完成的任务
4. **适当调整**：如果25分钟不适合你的工作，可以调整时长（需修改代码）
